# ÜniÜnity 🏫↔️🧑‍🏫↔️🏬

안녕하세요.👋 UniUnity는 대학교 인근상권을 한눈에 보여주는 서비스입니다.

소상공인 상가 정보와 제휴혜택 정보를 제공하고, 대학교와 학생, 인근상권 상인에게 커뮤니티를 제공해요!
